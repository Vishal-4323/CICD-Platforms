# CICD-Platforms
I created this repository for explore different CICD Platoforms

## Go Calculator

A simple calculator application built in Go.

### Features

- Addition
- Subtraction
- Multiplication
- Division

### Running the Application and Tests

To run the application, use the command:

```bash
go run main.go
go test ./calculator
```
